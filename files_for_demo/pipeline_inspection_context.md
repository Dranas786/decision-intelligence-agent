# Pipeline inspection context

Interpret the visualizations as follows:
- The 3D pipe scan shows the cleaned point cloud colored by signed radial deviation from the fitted nominal cylinder.
- The unwrapped defect map flattens the pipe surface into axial position by circumferential angle so localized dents appear as concentrated inward-deviation zones.
- The axial defect profile highlights where minimum deviation gets worse along the length of the pipe.
- The cross-section view compares a mid-pipe slice against the nominal circle to reveal out-of-roundness or inward damage.
- The ovality profile tracks broader shape distortion, while the dent summary focuses on localized inward defects.
- If fit RMSE is high, defect measurements should be treated as engineering-review candidates rather than final integrity decisions.
